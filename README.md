**3.사용 기술**
---
|제목|설명|
|-----------|------------------------|
|디자인 패턴|싱글톤 패턴을 사용하여 Manager 관리|
|Object Pooling|자주 사용되는 객체는 Pool 관리하여 재사용|
|UI 자동화|유니티 UI 상에서 컴포넌트로 Drag&Drop되는 일을 줄이기 위한 편의성|
